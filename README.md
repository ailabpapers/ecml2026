Experiments can be ran by installing the required libraries listed in requirements.txt and then running:

```
python Codes/{Dataset}/{Learning Setting}/{algorithm}.py
```

Where, Dataset choices are:
- "Solar Radiation"
- "Air Quality"

And, Learning Settings are:
- "Domain"
- "Task"

And, algorithms are:
- "a2er"
- "aclt"
- "clear"
- "cmr"
